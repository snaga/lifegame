# 技術スタック

本プロジェクトでは以下の技術スタックを利用します。

- **言語**: MicroPython
- **処理系**: PicoCalc custom MicroPython firmware
- **ライブラリ**:
  - `picocalc.display`: 画面描画ライブラリ
  - `picocalc.terminal`: テキスト表示ライブラリ
  - `picocalc.keyboard`: キーボード入力ライブラリ
  - `time`: 時間管理ライブラリ
- **開発環境**:
  - Thonny IDE または rshell を利用したシリアルコンソール経由での開発を想定
